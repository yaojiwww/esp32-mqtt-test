# esp32-mqtt-test
这是一个基于esp32和mqtt的物联网项目，主要用于测试mqtt的通信协议。

## 目录结构
esp32-mqtt-test/
├── esp32_code/
    ├── esp32_code.ino/    # 存放组件库
├── node_modules/        # 存放主程序
├── index.js/              # 存放静态文件
├── package.json         # 存放依赖包
├── package-lock.json       # 存放编译环境配置
├── README.md    # 项目说明文档
└── server.js    # 编译配置文件
